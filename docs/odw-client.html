Import into Figma (no plan upgrade)

The Figma MCP is capped at 6 calls/month on the Starter/View seat, so I can't push to the file directly. The html.to.design plugin converts these screens into editable Figma layers — text, frames, vectors — not flat images.

A · Browser-extension flow (best fidelity, works with localhost)

  1. In Figma: Plugins → html.to.design (install from Community if needed).
  2. Install the companion html.to.design browser extension (Chrome/Firefox).
  3. Open a screen from the grid above (each ?only=s0X URL is one clean frame).
  4. Click the extension → Send to Figma. The frame lands as editable layers.
  5. Repeat per screen. Tip: import ?only=s01 (the hero) first.

B · URL flow (if you expose a public URL)

  1. Make the prototype reachable: npx serve docs then a tunnel (e.g. ngrok http 3000), or host the file.
  2. In the plugin choose Import web page and paste the public …/odw-client-mockup.html?only=s01 URL.
  3. Repeat per screen.

C · Quick option

  1. Drag the 10 PNGs in docs/odw-client-screens/ straight onto a Figma canvas as reference frames.

Serve locally first: cd docs && python3 -m http.server 4319 → open http://127.0.0.1:4319/odw-client.html. Fonts are Gaegu / Inter / JetBrains Mono (Google Fonts) — the plugin fetches them automatically.
